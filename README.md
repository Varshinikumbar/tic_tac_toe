# Tic-Tac-Toe

A simple, interactive command-line **Tic-Tac-Toe** game built using Python. This project allows two players to play the classic game right from the terminal.

---

## 🚀 Features

* **Two-Player Mode:** Play locally with a friend alternating between 'X' and 'O'.
* **Input Validation:** Prevents players from selecting already occupied cells or entering invalid coordinates.
* **Win/Draw Detection:** Automatically detects and announces rows, columns, or diagonal wins, as well as tie games.
